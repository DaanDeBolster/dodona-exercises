Het Doppler

## Invoer

- Een tekst
- Een geheel `n`, waarvan je er vanuit mag gaan dat `n` groter is dan of gelijk is aan `1`. 


## Uitvoer

- Dezelfde tekts, maar `n` keer herhaald.

## Voorbeelden

```
>>> Hallo
>>> 2
HalloHallo

>>> Voorbeeld
>>> 6
VoorbeeldVoorbeeldVoorbeeldVoorbeeldVoorbeeldVoorbeeld
```
